## Features

- Kali Linux-like interface
- Fun commands
- Tab autocomplete
- Hotkeys
- Responsiveness
- Interactive UI

## Building from source

```sh
# 1. Clone the repo
git clone https://github.com/zake1god/zake1god.github.io.git
cd zake1god.github.io

# 2. Install the necessary dependencies
npm i

# 3. Run the app
npm start
```
